# BondCalc

**Калькулятор доходности облигаций с учётом инфляции** — десктопное приложение на Windows Forms для расчёта реальной (скорректированной на инфляцию) доходности облигаций к погашению с учётом купонного графика и амортизации.

---

## Возможности

- **Параметры облигации**: номинал, дата размещения, дата погашения
- **График амортизации**: ручное добавление/удаление/очистка или автогенерация разбиением номинала на N равных частей с динамическим предпросмотром суммы; мультиредактирование
- **Купонный график**: ручное добавление/удаление/очистка или автогенерация по периоду (пресеты 30/90/180 дней); генерация учитывает амортизации (уменьшающийся номинал); двухсторонняя синхронизация ставки (%) и суммы; мультиредактирование
- **Параметры сделки**: чистая цена, НКД, дата покупки, годовая ставка инфляции
- **Результаты** — номинальные (слева) и реальные/скорректированные на инфляцию (справа):
  - Грязная цена, доход от погашения, купонный доход, общий доход, общая доходность, годовая доходность
- **Расписание выплат**: таблица с колонками Дата, Тип, Номинал, Накопленный доход, Реальная сумма, Накопленный реальный доход (только чтение)
- **График накопленной доходности**: 3 сглаженных сплайна — номинальная доходность (зелёный), реальная доходность (синий), инфляция (красный); динамическое масштабирование оси Y
- **Экспорт**: расписание, параметры и результаты в отформатированный TXT-файл
- **Локализация**: английский (по умолчанию) и русский, переключение через меню «Язык»; сохраняется между сессиями
- **О программе**: информация о приложении со ссылкой на GitHub
- **Сброс**: сброс всех полей к значениям по умолчанию одной кнопкой

## Инструкция по использованию

### 1. Параметры облигации
Введите **номинал**, **дату размещения** и **дату погашения**.

### 2. Амортизации (опционально)
Если облигация гасится частями до погашения:
- Нажмите **Add** для ручного добавления строк или укажите количество частей и нажмите **Generate**, чтобы разделить номинал на равные платежи между датой размещения и погашения
- Выделите несколько строк, нажмите **F2** и отредактируйте одну ячейку — значение применится ко всем выделенным

### 3. Купоны
- Укажите **дату первого** и **последнего** купона, а также **период** в днях (или используйте кнопки 30/90/180)
- Введите **ставку (%)** или фиксированную **сумму** — они синхронизируются автоматически
- Нажмите **Generate** для автогенерации или **Add** для ручного добавления
- При наличии амортизаций сумма купона автоматически пересчитывается с учётом уменьшенного номинала

### 4. Сделка
Введите **чистую цену** покупки, **НКД**, **дату покупки** и ожидаемую **годовую инфляцию** (в процентах).

### 5. Расчёт
Нажмите **Calculate** — будут рассчитаны все результаты, таблица и график.

### 6. Экспорт
Меню **Export > As TXT** — сохранить отформатированный отчёт в файл.

### 7. Язык
Переключение между русским и английским через меню **Language/Язык**. Выбор сохраняется и восстанавливается при следующем запуске.
