# BondCalc

**Калькулятор доходности облигаций с учётом инфляции** — десктопное приложение на Windows Forms для расчёта реальной (скорректированной на инфляцию) доходности облигаций к погашению с учётом купонного графика и амортизации.

---

## Возможности

- Ввод параметров облигации: номинал, дата размещения, дата погашения, купонный график, график амортизации (частичного погашения номинала)
- Ввод параметров сделки: цена покупки, НКД, дата покупки
- Расчёт **номинальной** и **реальной (с поправкой на инфляцию)** доходности к погашению
- Корректировка на инфляцию с ежедневной капитализацией: `(1 + ставка / 365.25)^дни`
- **Таблица купонного графика** — дата, номинальная сумма, реальная сумма, накопленные итоги
- **График накопленной доходности** — 3 серии: номинальная доходность, реальная доходность, кривая инфляции
- Ручное добавление/удаление купонов и амортизаций, а также автогенерация по периоду (30/90/180 дней)
- Динамический предпросмотр суммы амортизации
- Локализация: английский (по умолчанию) и русский, переключение через меню «Язык»
